Ollama 설치 방법 (최초 1회)
1. https://ollama.com 에서 Ollama 설치 (Windows 지원)
2. 설치 후 cmd에서 모델 다운로드: ollama pull qwen2.5:7b
3. Ollama는 백그라운드에서 자동으로 실행됩니다

추천 모델 (한국어 번역 품질순)
• qwen2.5:7b — 기본값, 4.7GB, 한국어 최우수
• qwen2.5:14b — 더 높은 품질, 9GB
• gemma3:12b — Google 모델, 8GB
console.anthropic.com → API Keys에서 발급. .env 파일에 저장됩니다.

파일 경로
